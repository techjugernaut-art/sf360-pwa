StoreFront 360 Updates!

See what's new in SF 360 and what we're gradually rolling out!

May 2020

Release date: May 12, 2020

  • Mall Order Status Update
    • This enables StoreFront Mall users to update the status of orders coming in from the mall. The status update is done at the package level. The following statuses are available for them to switch to (ORDER PLACED, PROCESSING, READY FOR PICKUP, IN DELIVERY AND DELIVERED)
  • Delivery Pickup Location
    • This is a settings tab on the Shop Settings page that allows the user to set shop geolocation using google map. The longitudes and latitudes are required since they are needed for delivery. If not provided, the shop will not be listed on the mall.
  • Online Store Payment Request
    • This is a sub-action under SETTINGS to enable users with shops listed on the Mall, to submit their KYC documents for approval to accept payments on the mall.
  • Pay Subscription Enforcement
    • This ensures that all features on SF360 (Mall) are paid features. Users MUST be on premium to access this portal.
  • Product Image Upload
    • Allows users to add product images by clicking on Option Menu of the product and selecting Upload Image.
  • Forgot Password
    • This action is available on the Enter PIN page. This enables the user to reset their password.
  • Change Password
    • You can find this action under the user’s profile icon. This allows the user to change an old password to a new one.
  • Delete/Edit Banner
    • With this action available on the My Banner tab of the My Mall settings page, a user can Edit or Delete a slider Banner.
  • Print Mall Order
    • It's an action on the mall orders, that allows a user to print a mall order (Feedback from Amatar)
  • Bug fixes
    • Corrected “display” to “Display” on the onboarding page of mall customization, and on the My Mall settings page
    • Fixed appending of country again to existing phone number with country code on the reset password page. The happens when error happens on the server and user tries to submit the same form again
    • User not upgraded after subscription has been paid (Backend fix)
    • Password reset failure (Backend)
    • Product not listed on mall after all checklists have been exhausted (Backend)

April 2020

Release date: April 12, 2020

  • Signup / Onboarding/ Shop Creation
    • New users can create KudiGo StoreFront Account on SF360. They can onboard and make payment for subscription during the onboarding.
  • My Mall settings
    • This is part of the new My Mall menu introduced on SF360 to enable users customize their mall. They can perform actions like (update online address of mall, banner settings)
  • Mall Orders
    • This menu presents a page to display all orders coming to a users’ shop from StoreFront Mall. These orders have to be updated to DELIVERED for them to appear as sales orders in the reports.
  • Total Rebranding of SF360 to SF-Mall

March 2020

Release date: March 8, 2020

  • Total Supplier Stock Value on Product Page
    • This value represents the summation of all Supplier Price x Quantity in stock for all products and displays the Total Amount in the shop's default currency on the product page.
  • Total Unit Stock Value on Product Page
    • This value represents the summation of all Selling Price x Quantity in stock for all products and displays the Total Amount in the shop's default currency on the product page.
  • Download all products on Product Page
    • This is a sub-action under ACTIONS on the product page that allows a user to download all products of a selected shop.
  • Stock Record
    • This is a new tab introduced on the Product Detail page, displaying all the actions (eg. Decreased Stock, Increased Stock, Order Placed, etc) that occured on a product. This can filter by the Stock Update Activity and Date range.
  • Stock History Date Filter
    • This filter action is available on the Stock History tab on Product Details Page. This allows the user filter stock history within a date range.
  • Bug fixes
    • A screenshot of the Total Stock Value and Total Unit Value on product page

February 2020

Release date: February 8, 2020

  • Prepopulated columns for Product Upload
    • Initially the user had to map the Excel Sheet Columns to System Columns after uploading the template. This new update automatically maps the columns it can identify in the Excel Sheet and maps to the Server columns it can identify to be related.
  • Product Upload Event Tracking
    • This update currently helps track the various events that occur when uploading products (ie. Upload Template, Validate Template).
  • Expense Overview
    • This overview has been added on the dashboard that presents a horizontal bar chart that displays all total expenses according to category.
  • Requirement Information Tab
    • This tab has been added on the payment request page that highlights all the details (requirements) that are needed to successfully complete a payment request submission process.
  • Payment Request Tooltips
    • These tooltips have been added on the various when submitting a payment request.
  • Preview Product Barcodes
    • This feature allows the preview of Barcodes for all Products. The barcodes can be printed as per the number of Barcodes one wants to print per product.
  • New Assets
    • These assets include Logo and Favicon.

  • Google Analytics

  • Payment Request

June 2019

Release date: June 6, 2019

  • OTP Login
    • After entering your Phone Number and PIN, you will receive an OTP code in an sms to confirm your login (We are still working on how to remember your device hence not sending you OTP at all times)
  • Business Overview Dashboard
    This module gives an overview for
    • Sales Orders
    • Sales Margin
    • Customers that visited the shop
    • Suppliers
    • Sales Orders over a trend (Line graph)
    • Payment methods over a fraction (Doughnut graph)
    • These overviews are affected by a Date Range and Shop filter, which means you can also filter by All Shops, or a specific shop in your Storefront Account.
  • Reports Module
    This modules provides both readable and downloadable reports on
    • Sales Summary (Can be filtered by Date Range, Payment Method, Order Status, and Shop)
    • Sales Orders (Can be filtered by Date Range, Payment Method, Order Status, and Shop)
    • Sales Margin (Can be filtered by Date Range, Payment Method, and Shop)
  • Agents Module
    • This module provides an overview of the performance of agents.
  • Products Module
    • Products module provides a list of Products available in the shop. Admin can filter by shop.
  • Product Upload
    • This is an action on the Product Page where an Admin can use this module to perform bulk upload of products into a specific shop.
  • Customers Module
    • Admin can view a list of Customers that visited his shop and the number of times they visited.
      Whether they are owing or not; amount they owe and the next payment date
      Can be filtered by Shop and Owing Customers.
  • Suppliers Module
    • Provides an overview into the suppliers of a shop. Can also be filtered by Shop.
  • Intellisight
